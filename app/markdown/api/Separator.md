## 分隔符类

> 分隔符类 `use Kingbes\Libui\Separator;`

<br>

#### 创建水平分隔符

静态方法`createHorizontal`

- 参数
    - 无
- 返回
    - `CData` 水平分隔符

```php
public static function createHorizontal(): CData
```

<br>

#### 创建垂直分隔符

静态方法`createVertical`

- 参数
    - 无
- 返回
    - `CData` 垂直分隔符

```php
public static function createVertical(): CData
```
